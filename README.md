Compilation Instructions


    Java:
        cd Java/
        javac parser.java
        java parser ../translations/translation

    C++:
        cd cpp/
        c++ parser.cpp 
        a.out ../translations/translation        

    Perl:
        cd perl/
        perl parser.pl ../translations/translation

    Fortran:
        cd fortan/
        gfortran parser.f90
        parser ../translations/translation

    Ada:
        gnatmake parser.adb
        parser ../translations/translation
